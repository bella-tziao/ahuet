BASIC

Основи класичного масажу, діагностика в масажних техніках

Ідеальний для старту та вдосконалення навичок. Вивчення анатомії, ергономіка майстра, постановка рук, базові та продвинуті техніки масажу по тілу.

Програма курсу (44 години)

1. Анатомія масажиста (6 годин)

Вивчення мʼязової топографії, функції мʼязів:

  • Спина, Таз, Ноги
  • Голова, обличчя, Шия

2. Шкіра та засоби (2 години)

Будова та функції шкіри, рецепторний апарат. Розбір засобів для масажу.

3. Лімфатична система (6 годин)

Анатомія, функції, лімфодренажний масаж. Абдомінальний масаж.

4. Антицелюлітний блок (4 години)

Поняття про ожиріння, робота гепатобіліарної системи. Будова жирової клітини.

5. Нервова та судинна системи (6 годин)

  • Робота при судинній сітці
  • Вплив масажу на судини
  • Нейроседативні та релакс масажі

6. Масаж спини (6 годин)

Діагностика: трігери, міозит, спаєчні процеси. Атонія та гіпертонус.

7. Інші зони (14 годин сумарно)

  • Масаж шиї, голови, обличчя (4 год)
  • Грудні мʼязи, діафрагма, живіт, руки (4 год)
  • Стопи, ноги, мʼязи тазового поясу (6 год)

PROFI

Скульптор Тіла

Просунутий рівень для корекції фігури. Антицелюлітні програми масажу, бамбукові палички, блейди, динамічні техніки та лімфодренаж. Вакуумний масаж, комбінація з апаратними методиками та ін'єкціями ліполітичних препаратів. Абдомінальний масаж живота.

Програма курсу (20 годин)

1. Фізіологія схуднення (6 годин)

Анатомія та фізіологія роботи лімфатичної та гепатобіліарної систем.

2. Робота з жировою клітиною (6 годин)

Техніки для роботи з адипоцитом. Принципи зменшення кількості та покращення якості жирової клітковини.

3. Комбінована терапія (2 години)

Поєднання ручного масажу з апаратними та ін'єкційними методиками для максимального результату.

4. Протоколи та діагностика (2 години)

Розгляд шаблону для складання індивідуальної програми корекції фігури під клієнта.

5. Запуск метаболізму (4 години)

Техніка точково-абдомінального масажу як старт для запуску процесів схуднення та обміну речовин.

Тантрична техніка, спрямована на глибоке опрацювання чоловічої енергії, зняття сексуальної напруги та зміцнення здоров'я.

Медитативний та надзвичайно дбайливий масаж, що допомагає жінці з'єднатися зі своїм тілом та зцілити емоційні травми.
